## 🚀 How to Get Started (5 Minutes)

### Step 1: Install React Native Project
```bash
npx create-expo-app hogu-mobile
cd hogu-mobile
npm install
```

### Step 2: Copy Mobile App Files
```bash
# Copy from /Users/sdheeraj/VS-new/ to hogu-mobile/
cp theme.js api.js App.js .
cp LoginScreen.js screens/
cp DashboardScreen.js screens/
cp ActivityDetailScreen.js screens/
cp ActivityCard.js components/
cp TrackersScreen.js screens/
cp SettingsScreen.js screens/
```

### Step 3: Install Dependencies
```bash
npm install @react-navigation/native @react-navigation/bottom-tabs
npm install react-native-gesture-handler react-native-reanimated
npm install axios moment @react-native-async-storage/async-storage
```

### Step 4: Start Development
```bash
npx expo start
# Scan QR code with Expo Go app on your phone
```

### Step 5: Test Flows
- Sign up with email
- Login with credentials
- View activities
- Tap activity to see details
- Switch to Trackers tab
- Go to Settings
- Logout

## 🔗 Backend Integration Ready

The mobile app is ready to connect to your backend. Expected API endpoints:

```javascript
// Authentication
POST /auth/login → { token, user }
POST /auth/register → { success }
POST /auth/logout → { success }

// Activities (Already working with Strava)
GET /api/activities → [ activities ]
POST /api/refresh → [ activities ]

// User Management
GET /user/profile → { user }
PUT /user/profile → { user }
POST /user/password → { success }

// Trackers
GET /user/trackers → [ trackers ]
POST /trackers/{type}/auth → { oauth_url }
POST /trackers/{id}/disconnect → { success }
```

All API calls are abstracted in `api.js` - change URLs in one place!

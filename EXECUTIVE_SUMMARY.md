## What HOG-U Does

### In One Sentence
HOG-U automatically fetches fitness data from connected trackers, enriches it with intelligent analysis, and generates personalized meal recommendations on a live dashboard.

### In One Paragraph
When a user connects their Strava account, HOG-U automatically fetches their recent activities including distance, elevation, time, and location. It then intelligently processes this data by: calculating intensity metrics, converting time zones, mapping coordinates to actual location names (e.g., Lalbagh), fetching historical weather conditions, and generating personalized meal recommendations based on the intensity of the workout, environmental conditions, and meal timing. All of this happens automatically without any manual work, displayed on a beautiful dashboard at http://localhost:8080, and is ready to order from Zomato.

## How It Works: 3-Step Process

### Step 1: Authentication (One-Time)
```
User clicks "Connect with Strava"
    ↓
HOG-U generates OAuth login link
    ↓
User logs in and authorizes
    ↓
System gets permanent access token
    ↓
✅ Ready for auto-fetch
```

### Step 2: Auto-Fetch (Automatic)
```
System calls Strava API
    ↓
Fetches last 5 activities
    ↓
Extracts: name, type, distance, elevation, time, coordinates
    ↓
✅ Data ready for processing
```

### Step 3: Intelligent Processing (Automatic)
```
Calculate intensity index = elevation / distance
Map coordinates → location name
Convert timestamp → local timezone
Fetch historical weather
Analyze intensity + humidity → recommendation level
Generate meal suggestion
Create Zomato search link
    ↓
✅ Ready for dashboard display
```
